css 출력 및 박스 관련 속성

display, visibility, opacity, width, height, overflow, margin, padding, border 등

모든 속성의 값에 initial(초기값), inherit(부모요소로 부터 상속)

화면출력 속성

display

태그마다 화면 출력 모드가 block, inline, inline-block 등으로 구성됨.

inline

속성요소 : span, a, strong, em, ins 등

특징 : 한 줄 안에 자동 줄바꿈이 될 때까지 배치됨 크기지정을 할 수 없어 width, height, line-height, text-indent, text-align padding-top, padding-bottom, margin-top, margin-bottom 등의 속성을 지정한다고 하더라도 결과에 반영되지 못 함.

block

속성요소 : div, h1~h6, p, section, article, header, footer ...

특징 : 자동으로 다음 줄에 배치가 되며, 크기와 관련된 속성 지정 가능.

부여 가능한 속성 값 : inline | block | inline-block | none

부여 가능한 속성 값 2 : contents, flex, grid, box, table, inline-flex, inline-grid, inline-table, list-item, run-in, table-caption, table-column, table-row, table-cell, table-header-group, table-footer-group, table-column-group, table-row-group

visibility(가시속성)

보일지 보이지 않을지를 나태내는 속성

visible | hidden | collapse

opacity

불투명 정도를 나타내는 속성

0~1까지의 실수 또는 정수로 수치를 표시하며, 0은 투명, 1은 불투명임

해당 요소의 글자와 모든 콘텐츠(배경 이미지 및 배경색)를 불투명 또는 투명하게 지정 가능함.

width(너비속성)

가로(폭=너비)의 크기를 지정하는 속성

auto | length | %

length는 px, pt, pc, cm, mm, in(ch), vw, vimin, vmax 등의 단위로 지정이 가능

기본 화면 출력 속성이 블록요소 또는 인라인블록요소 등에만 크기지정이 가능함.

height(높이속성)

세로(높이)의 크기를 지정하는 속성

overflow(흘러넘침속성)

콘텐츠의 양이 지정한 크기(폭/넓이)보다 더 클 경우 지정

auto|visible|hidden(흘러넘친 것은 표시x)|scroll|clip

세부속성 : overflow-x(가로), overflow-y(세로)

margin(바깥여백)

현재 테두리(콘텐츠) 영역의 바깥 여백으로서 여러 콘텐츠와 콘텐츠 사이의 여백

auto | length | %

length는 px, pt, pc, cm, mm, in(ch), vw, vimin, vmax 등의 단위로 지정이 가능

margin 값 : margin = 위/아래, 좌/우 margin = 상 우 하 좌

margin 30px auto - 가운데 배치 할 때 많이 씀.

통합 속성 먼저 걸고, 아래에 세부속성.

세부 속성 : margin-top, margin-bottom, margin-left, margin-right

padding(안쪽여백)

테두리와 콘텐츠 사이의 여백

auto | length | %

length는 px, pt, pc, cm, mm, in(ch), vw, vimin, vmax 등의 단위로 지정이 가능

padding 값 : padding = 위/아래, 좌/우 padding = 상 우 하 좌

세부속성 : padding-top, padding-bottom, padding-left, padding-right

통합 속성 먼저 걸고, 아래에 세부속성, 지정한 폭/높이에 padding과 border 값을 더한 값이 실제 콘텐츠의 크기가 됨. (왼 border + 왼 padding + 콘텐츠 + 우 padding + 우 border == 실제크기)
