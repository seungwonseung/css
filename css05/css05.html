배경 관련 속성

background-attachment, bgd-blend-mode, bgd-origin, bgd-clip, bgd-color,bgd-image, bgd-position, bgd-repeat, bgd-size 등이 있음.

bgd-color를 제외한 나머지 속성은 배경이미지에 대한 설정임. 또한, 배경 이미지는 여러 이미지를 겹치게 할 수 있음.

background : bg-color bg-image bg-position/bg-size bg-repeat bg-origin bg-clip bg-attachment

background 통합 속성은 여러 속성을 한 꺼번에 지정할 수 있지만 각 세부 속성의 순서를 지키면서 설정해야 함. 특히, 위치(position)과 크기(size)는 필수적으로 순서를 지켜야 함.



배경색(color)

단색의 경우

background : rgba(221, 120, 18, 0.8);

background-color : rgba(221, 120, 18, 0.8);



그라디언트의 경우

background : linear-gradient(135deg, deeppink, deepskyblue)[어느방향으로 진행을 할 지.deg]

background : radial-gradient(red, orange, yellow, green, blue)



필터의 경우

filter : url(./images/ogu.jpg)

filter : blur(5px)

filter : contrast(200%)

filter : grayscale(80%)

filter : hue-rotate(90deg)

filter : drop-shadow(10px 15px 20px red)



배경 이미지 설정

background-image : url(파일의 위치와 파일명);

background-image : url(./images/ogu.jpg)

background-image : url("./images/오구.jpg") + 띄어쓰기도



배경 위치 설정

background-position : 가로위치 세로위치

background-position : left center
키워드일 경우는 가로 방향은 left/center/right
세로 방향은 top/center/bottom

background-position : 10% 100px
특정 숫자가 기입될 경우는 단위를 반드시 명기를 해야함.

background-position : -200px -400px 아이콘리스트이미지인 경우 해당 이미지의 위치 값에 따라 다른 아이콘 나오게 할 수 있음



배경 크기

background-size: auto | length | cover | contain

auto : 배경 이미지 크기 그대로 표시

length : %, px 등의 단위를 숫자와 그 숫자 뒤에 병기해서 사용

cover : 가로 폭의 맞추어 높이는 부분은 감안하지 않는 방법

contain : 해당 콘텐츠의 크기에 맞추어 조절하는 방법

background-size: 가로크기 세로크기



배경 반복 속성

background-repeat : repreat | repeat-x | repeat-y | no-repeat

해당 요소의 크기가 배경 이미지의 크기보다 클 경우에만 유효하다.

repeat : 가로 방향과 세로 방향 모두 적용됨

repeat-x : 가로 방향만 적용됨

repeat-y : 세로 방향만 적용됨

no-repeat : 반복하지 않고, 한 번만 출력



배경 시작점 속성

background-origin : padding-box | border-box | content-box

배경 이미지가 어디 부터 시작할지 결정

padding-box : padding 부분부터 이미지가 시작됨.

border-box : border 부분부터 이미지가 시작됨.

content-box : content 부분부터 이미지가 시작됨.



배경 적용 범위 속성

background-clip : padding-box | border-box | content-box

배경 이미지가 어디 까지 표시될 것인지 결정

padding-box : padding 영역까지 이미지가 표시됨

border-box : border 영역까지 이미지가 표시됨

content-box : content 영역까지 이미지가 표시됨



배경 이미지 스크롤 속성

background-attachment : scroll | fixed | local

배경 이미지가 마우스 스크롤시에서 스크롤 유무를 결정

scroll 또는 local : 일반 콘텐츠 처럼 스크롤된다.

fixed : 배경 이미지가 화면에 고정되고, 스크롤 되지 않는다.
